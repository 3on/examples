## Setup

```bash
git clone https://github.com/electron-react-boilerplate/examples
cd examples

# If you don't have lerna, run `npm i -g lerna@latest` or `yarn global add lerna@latest`
lerna bootstrap

# cd to an example you'd like to run
cd sqlite
# Development
yarn dev
# Package for production
yarn package
```
